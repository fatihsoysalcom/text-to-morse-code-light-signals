# Text to Morse Code Light Signals

This example demonstrates how a plain text message can be converted into Morse code, a foundational concept for digital communication. It then simulates the transmission of this Morse code using "optical signals" represented by short (dot) and long (dash) flashes. This illustrates how information can be carried by varying patterns of light, mirroring the principles of early optical communication.

## Language

`python`

## How to Run

1. Save the code as `main.py`.
2. Run from your terminal: `python main.py`
